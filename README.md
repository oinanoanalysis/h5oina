# Oxford Instruments NanoAnalysis HDF5 File Specification

This repository contains the specification for the Oxford Instruments NanoAnalysis HDF5 file format (_.h5oina_).
The H5OINA file format is based on [Hierarchical Data Format 5 (HDF5)](http://www.hdfgroup.org) and has been designed to exchange microanalysis data within the [Oxford Instruments](https://oxinst.com/nano)â€™ AZtec ecosystem as well as with third-party software and libraries.
